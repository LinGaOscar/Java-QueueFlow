# QueueFlow 系統教學筆記

本目錄為 QueueFlow 高併發排隊系統的教學文件，適合搭配源碼閱讀使用。

| 文件 | 內容摘要 |
|------|---------|
| [01-架構總覽](./01-架構總覽.md) | 技術選型理由、分層架構圖、雙資料庫分工說明 |
| [02-重點對照](./02-重點對照.md) | Redis vs DB、INCR vs 時間戳、Lua 原子性、分散式鎖、WebSocket vs 輪詢等核心概念比較 |
| [03-系統解析](./03-系統解析.md) | 狀態機、joinQueue 完整步驟、admitQueue 流程、WebSocket 推播架構、活動生命週期 |
| [04-周邊機制](./04-周邊機制.md) | 限流、防重複入列、逾時排程、稽核日誌、Schema 管理、全域例外、前端 XSS 與命名禁忌 |

## 建議閱讀順序

1. **先看 01** 建立整體認識
2. **再看 03** 跟著程式碼追流程（搭配 `QueueService.java` / `EventService.java`）
3. **回頭看 02** 理解每個設計決策為何這樣做
4. **最後看 04** 了解各周邊機制的細節與教學注意事項
